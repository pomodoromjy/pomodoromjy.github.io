常用 Git 操作---将本地文件提交到 GitHub 上
1. 初始化项目文件夹
进入目标文件夹，执行 git init
2. 将文件添加到暂存区里，执行 git add .
3. 将暂存区的文件提交到远程仓库中去，执行 git commit -m '本次添加说明'
4. 关联到远程库，执行 git remote add origin 报错则先执行 git remote rm origin,再执行 git remote add origin https://github.com/pomodoromjy/pomodoromjy.github.io.git
5. 获取远程库与本地合并（如果远程库为空则可忽略此步骤）git pull --rebase origin master
6. 将本地库的内容推送到远程库 git push -u origin master
