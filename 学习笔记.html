学习笔记
1. 机器学习泛化力：不断从经历及数据中汲取教训，从而预知未来的能力。
2. 监督学习常用于对未知表现的预测，而无监督学习常用于对事物本身特性的预知。
3. python 常用数据类型：
(1) python 是大小写敏感的语言，布尔值为 True 和 False.
(2)元组 Tuple：由一系列数据类型可不同的数据组成的序列,可使用元组索引来查看。
  eg.t = (1,'abc',0.1)
(3)列表 List:列表在功能上基本与元组相似，只是课表允许使用者在访问时对其进行修改，元组则不然。
  eg.L = [1,'abc',0.1)]
(4)字典 Dictionary:由多组键值对组成，只是对于键的数据类型没有要求。
  eg.d = {1:'1','abc':0.1},d[1] 的值为'1'
4. python 常见运算类型
(1)成员运算 in：判断某个元素是否存在于元组、列表以及某个键是否存在云字典中，返回类型为 boolean.
5. python 函数：由关键字 def 来定义
