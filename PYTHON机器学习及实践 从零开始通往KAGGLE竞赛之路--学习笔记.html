3. python 常用数据类型：
(1) python 是大小写敏感的语言，布尔值为 True 和 False.
(2)元组 Tuple：由一系列数据类型可不同的数据组成的序列,可使用元组索引来查看。

  eg.L = [1,'abc',0.1)]
(4)字典 Dictionary:由多组键值对组成，只是对于键的数据类型没有要求。
  eg.d = {1:'1','abc':0.1},d[1] 的值为'1'

5.16
1. 监督学习--分类学习
(1) 线性分类器：假设特征与分类结果之间存在着线性关系，通过计算每个特征与其权重的乘积来帮助类别决策。逻辑斯蒂回归模型。
(2) 支持向量机：根据所有训练数据的分布情况，选择所有线性分类器中最佳的那个。其中，决定直线位置的并不是所有训练数据，而是训练数据中空间位置最小的两个数据决定的，这两个数据点就叫做“支持向量”。
(3) 朴素贝叶斯：首先，贝叶斯公式描述的是这么一个事情：新事物出现后事件 A 的概率 = A 原本的概率 X 新事物出现后带来的调整。“朴素贝叶斯”则是贝叶斯分类中最简单、最常见的分类方法，“朴素”一词代表假设各个特征之间相互独立。
(4) K 近邻：对于某个待分类样本，选取特征空间中距离其最近的 K 个已分类样本作为依据进行分类决策。与其他分类模型相比，该模型无参数预测阶段，因此开销大。
(5) 决策树。对非线性关系进行描述。
2. 数据预处理函数：fit_transform() 和 transform()
  fit_transform():fit to data,then transform it.先进行数据拟合，再转化成其标准形式
  transform():perform standardization by centering and scaling.通过寻找其中心和缩放实现标准化。

3. 机器学习中数据归一化方法
(1) 0 均值化归一：即均值为 0，方差为 1。避免某一分量数值太大而将其他分量掩盖掉（要求原始数据服从高斯分布）。
(2) 线性函数归一化：实现原始数据的等比例缩放
