PHP 学习笔记
一、PHP 基础
1. global 用于声明全局变量,当在局部变量前使用该关键字时，该局部变量会变成全局变量
2. static 当函数执行后，会删除所有变量，若使用 static 关键字，则可以保留该变量上次的值
3. var_dump() 返回变量类型及其值
4. strpos() 该字符串函数用于检索字符串内指定的字符或文本，若检索成功，则会返回匹配的首个字符串的位置，否则则返回 false
5. "." 字符串连接 ".=" 字符串连接并赋值
6. 换行 echo 内容."< br >"
7. foreach 用于遍历数组（关联数组）中的每个键值对
8. php 中 array() 函数用于创建数组; count() 函数用于返回数组的长度
9. $_REQUEST['name'] 用于收集 HTML 表单中名为 name 的表单值
二、PHP 与 Mysql
1. mysql_connect('参数一','参数二','参数三');函数 与数据库建立连接
参数一：数据库地址
参数二：数据库用户名
参数三：数据库用户密码
返回值：1. 连接成功时，返回 MySQL 连接标志符,起定位作用，方便关闭数据库连接；2. 连接失败时，返回 false
2. mysql_close()函数 关闭数据库连接
3. mysql_select_db('参数一','参数二') 选择数据库 当选择成功时返回 true，否则返回 false
参数一：需要连接的数据库连接标志符
参数二：需要连接的数据库名称
4. myqsl_query() 执行一条 MySQL 语句
