Logistic 回归应用——疝马死亡率预测
一、对数据进行预处理
1. 特征缺失值的处理
因为数据集中含有缺失值，本例中我们选择用 0 来代替缺失值，原因有二：
(1). 根据回归系数的更新公式 weight = weight + alpha * error * dataMat
如果 dataMat 对应的某个特征值为 0 ，那么 weight = weight，即回归系数不会更新
(2). 根据 sigmoid(0) = 0.5,它对结果没有任何倾向性，因此也不会对预测结果造成任何影响
